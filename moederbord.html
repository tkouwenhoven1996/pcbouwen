2. geheugen

 

Werkgeheugen (RAM) is nodig om een PC vlot te kunnen laten werken. RAM slaat alle bestanden die op dat moment gebruikt worden tijdelijk op. RAM is velen malen sneller dan opslagapparaten als hardeschijven of ssd's waardoor er sneller gewerkt kan worden met programma's.

RAMsoorten:

De meest gebruikte soort RAM is DDR3. Bijna alle moederborden ondersteunen dit en heeft een goede prijs/performance ratio. DDR4 geheugen is nieuwer maar ook duurder en zorgt niet voor een groot prestatieverschil. Vaak is het meest goedkope DDR3 RAM van een bekende fabrikant een goede koop. Let wel op de kloksnelheid van het geheugen. 1600MHz is een standaard snelheid en meestal de beste koop. Lagere kloksnelheden zorgen voor een lagere performance, al is dit verschil miniem.

 

Hoeveel RAM heb ik nodig?

 

Gaming:

Als je wil gamen is 4GB RAM geheugen ongeveer het minimum, 8GB is echter de sweet-spot. Veel geheugen zorgt meestal niet voor een prestatiewinst, echter kan te weinig geheugen zorgen voor een groot prestatieverlies.

Video-/fotoediten:

Bij video-/fotoediten is 8GB ook voldoende, echter kan 16GB of meer ook worden gebruikt. Als er meer RAM aanwezig is gebruikt het systeem meer RAM. Meer RAM bij video-/fotoediten zorgt voor snellere laadtijden een een betere overall experience.

Multimedia:

Bij multimedia is 2GB voldoende. Wil je echter veel applicaties tegelijkertijd open kunnen hebben staan is 4GB aan te raden.
